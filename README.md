# xmmxovo

A personal CLI dashboard for xmmxovo.

## Install

```bash
npm install -g xmmxovo
```

Or run directly with npx:

```bash
npx xmmxovo
```

## Usage

```bash
xmmxovo
xmmxovo about
xmmxovo site
xmmxovo social
xmmxovo help
```

## Commands

### `xmmxovo`
Show the main dashboard, including:
- welcome message
- profile summary
- site
- channel overview
- business contact

### `xmmxovo about`
Show the profile block.

### `xmmxovo site`
Show the website block.

### `xmmxovo social`
Show detailed social media links.

### `xmmxovo help`
Show the command list.
